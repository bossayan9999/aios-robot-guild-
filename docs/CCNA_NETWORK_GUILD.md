# CCNA Network Guild

The Network Guild is an interactive, browser-only CCNA learning workspace.

## Included

- Clickable router, Layer 3 switch, access switch, VLAN, server and endpoint topology.
- IPv4 subnet calculator for `/0` through `/32`.
- VLAN, subnetting, routing and troubleshooting quests with local XP rewards.
- Safe Cisco IOS command reference.
- Desktop and mobile layouts.

## Safety boundary

The first release is a simulator. It does not contact, scan or configure a real network device. Future connections must require an explicit device allowlist, read-only credentials by default, command preview, owner approval, time limits and complete audit logging.

The example public WAN address uses `203.0.113.0/24`, which is reserved for documentation.

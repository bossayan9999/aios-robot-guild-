# v0.6 — CyberScool Connectors and Skills

Status: Completed on feature branch

## Delivered

- Versioned connector and skill manifests for GitHub, Cloudflare, Supabase, Gmail, Calendar, Drive, Slack, Notion, model providers, MCP, REST, OAuth, and API-key integrations.
- Opaque server secret references, hashed expiring OAuth state, tenant/task grants, action classification, approval enforcement, health checks, and revocation.
- GitHub and Cloudflare live verification adapters that require credentials, scopes, provider identity, and health. Missing credentials produce `ERROR`.
- Live internal MCP verification and scoped read invocation for deterministic local validation.
- Rate limiting, bounded retries, timeout, circuit breaking, target/resource checks, idempotency, redacted summaries, and audit events.
- Integrations UI driven exclusively by authoritative connector instance state.

Provider credentials were not configured or exposed during validation. Provider-specific adapters not yet implemented remain `NOT_CONFIGURED`.

# v0.5 — CyberScool Specialist Runtime

Status: Validated on feature branch

## Delivered

- Ten typed, versioned built-in specialist definitions with schemas, skills, tools, connectors, runtimes, data scope, risk, budgets, approval rules, prohibitions, reviewers, and evaluation suites.
- Workspace-scoped skills, capability grants, evaluations, performance events, suspensions, revocations, and execution contracts.
- Independent sandbox evaluation, explicit approval for privileged grants, expiring authority, and cascading cancellation on revocation.
- Specialist Team rendering from authoritative evaluation and grant records.

## Boundaries

Execution contracts authorize bounded future work but do not dispatch commands or claim runtime health. Actual runtime enrollment and execution are Phase 7. Production deployment and merge approval are outside this release.

# Production Backlog

- Add organization/workspace isolation and role/permission test suites.
- Add authentication, approval, connector, runtime, terminal, and cost rate limits.
- Add secret, dependency, container, migration, and supply-chain scanning.
- Add OpenTelemetry, service-level objectives, alerting, incident response, and immutable audit export.
- Exercise backup restore, disaster recovery, credential rotation, runtime revocation, and deployment rollback.
- Require staging evidence, production approval, post-deployment verification, and an independent security review.

Production readiness is a completion-gated evidence decision, not a release label.
